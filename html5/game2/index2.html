Objective:

  • The objective of the game is to get the highest score you can get.
  • Number you get and the position they will appear are random and they can be move up if it is not at the top most blocks, move down if it is not at the bottom most blocks, move right if it is not at the right most blocks and move left if it is not at the left most blocks.
  • The score incease based on the total number you have on the board currently.
  • The game gets over when all the blocks gets filled up.


Logic Design and Implementation:

  • Made a separate class to make grids.
  • Write proper logics for move number blocks on the hit of "arrow keys" key.
  • Maintain a separate Environment class which will create new number and a class to add them and change their color based on the number.
  • Used proper naming of variable and classes for easy understanding.
  • Create logics to determine when the game is over.
